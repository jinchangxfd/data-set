
Project: OTA BootLoader Tx
Ver: OTA_BootLoader_Tx  V1.0.0
描述：主机通过24L01模块对从机进行无线升级。
主机接收固件方式：UART
从机接收固件方式：NRF


/**********硬件资源***************
MCU: STM32F103VET6
SRAM:64KB FLASH:512KB
无线模块：NRF24L01 
**********************************/

/**********程序设计****************
1，接收固件占用SRAM:	45KB（SRAM）
2，bootloader占用： 	10KB (FLASH) 预留：	20KB（FLASH）
**********************************/

/**********bug调试修复************

bug1：设置UART为115200，串口检测43000才能读取
	  设置HSE时钟为8MHz，问题解决

bug2：2个NRF24L01通讯出现掉包。
	  通过串口接收数据，发现接收端接收到数据有时出现没有应答位情况。
	  解决方案：在发送的数据前添加数据包的ID号，以发射端的ID号为准

**********************************/

/********NRF串口群组升级指令*******
1，0xff 接收固件  
2，0xfe 获取接收端ID
3，0xfd 更新固件
**********************************/
